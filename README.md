# 利用Python进行数据分析

### 运行环境
- python3.7


# 深度学习

### Tensorflow2.0安装
CPU版本安装：
```
pip install tensorflow
```

GPU版本安装(需要配置CUDA10)：
```
pip install tensorflow-gpu

```
> 安装tensorflow-gpu，python<=3.7

### 机器学习流程
1. 数据获取
2. 特征工程
3. 建立模型
4. 评估与应用

### 特征工程的作用
1. 数据特征决定了模型的上限
2. 预处理和特征提取是最核心的
3. 算法与参数选择决定了如何逼近这个上限

> 数据库样例：CIFAR-10，百度下载
